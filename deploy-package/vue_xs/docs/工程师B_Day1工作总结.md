## 💡 遇到的问题和解决方案

### 问题1：PowerShell不支持 `&&` 命令
**解决方案：** 分步执行命令，每个命令单独运行

### 问题2：首次创建项目时不熟悉Vite
**解决方案：** 参考官方文档，使用 `npm create vite@latest`

### 问题3：Tailwind配置
**解决方案：** 创建独立配置文件，引入Tailwind指令
